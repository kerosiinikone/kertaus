# ChatGPT kertausaikataulut

> Hyödyntää ChatGPT:tä kertausaikataulujen tuottamiseen lukion aihealueista

## Dev Environment

### Server

```bash
npm run start
```

### Client

```bash
cd .\client\
npm run dev
```
